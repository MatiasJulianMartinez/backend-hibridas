Documentación de la API - Mundo Funko

Endpoints Publicos para Ruta Usuarios

  • GET - /api/users - Obtener todos los Usuarios
  • GET - /api/users/{id users} - 680f936a420c9d04cc7c3c58 - Obtener un Usuario por ID
POST - /api/users/register Registrar un nuevo Usuario

Content-Type: application/json - Tiene validacion - Devuelve Token.

Ejemplo de cuerpo (body) del POST:

                {
                  "name":"Matias Emanuel Vallejos",
                  "email":"matiasvallejos@gmail.com",
                  "password":"123456"
               }
POST - /api/users/login Logeo de Usuario

Content-Type: application/json - Tiene validacion - Devuelve Token.

Ejemplo de cuerpo (body) del POST:

                        {
                          "email":"matiasvallejos@gmail.com",
                          "password":"123456"
                       }
PUT - /api/users/{id-users} Modificar un Usuario

Content-Type: application/json - Requiere Token y Validacion.

Ejemplo de cuerpo (body) del POST:

                        {
                          "name":"Matias Emanuel Vallejos modificado",
                          "email":"matiasvallejos@gmail.com",
                          "password":"123456"
                       }
DELETE - /api/users/{id-users} Eliminar un Usuario

Content-Type: application/json - Requiere Token.

Endpoints Públicos Ruta Funkos

  • GET - /api/funkos - Obtener todos los Funkos
  • GET - /api/funkos/{id funko} - 680f94c7420c9d04cc7c3c7f - Obtener un Funko por ID
  • GET (filtro por categoría) - /api/funkos?categoriaId=680bab4e23896b0d0e324ab6 - Obtener Funkos por filtro de Categoría
  • GET (filtro por precio) - /api/funkos?precio=5000 - Obtener Funkos por filtro de Precio
  • GET (filtro por tipo) - /api/funkos?tipo=replica - Obtener Funkos por filtro de Tipo de Funko
  • GET (filtro por nombre) - /api/funkos?nombre=goku - Obtener Funkos por su Nombre
POST - /api/funkos Crear un nuevo Funko

Content-Type: application/json - Requiere Token y validacion.

Ejemplo de cuerpo (body) del POST:

        {
          "nombre": "Funko Pop Harry Potter",
          "categoriaId": "680d28706d7776fb08d81013",
          "precio": 25000,
          "tipo": "Replica",
          "stock": 10
      }
PUT - /api/funkos/{id-funko} Modificar un Funko

Content-Type: application/json - Requiere Token y validacion.

Ejemplo de cuerpo (body) del PUT:

              {
                "nombre": "Funko Pop Harry Potter with Ron",
                "categoriaId": "680d28706d7776fb08d81013",
                "precio": 25000,
                "tipo": "Replica",
                "stock": 10
            }
DELETE - /api/funkos/{id-funko} Eliminar un Funko

Requiere Token

Endpoints Públicos Ruta Categorias

  • GET - /api/categorias - Obtener todas las Categorías
  • GET - /api/categorias/{id categoria} - 680d28706d7776fb08d81013 - Obtener categoría por ID
  • GET (filtro por descripcion) - /api/categorias?descripcion=Peliculas - Obtener Categorias por filtro de Descripcion
  • GET (filtro por tags) - /api/categorias?tags=Stranger Things - Obtener Categorias por filtro de Tags
  • GET (filtro por Pais de Origen) - /api/categorias?paisOrigen=Japon - Obtener categorias por filtro de Pais de Origen
  • GET (filtro por material) - /api/categorias?material=Vinilo - Obtener categorias por filtro de Material
  • GET (filtro por nombre) - /api/categorias?nombre=Anime - Obtener categorias por su Nombre
POST - /api/categorias Crear una nueva Categoría

Content-Type: application/json - Requiere Token y validacion.

Ejemplo de cuerpo (body) del POST:

            {
              "nombre": "Funko Pop Anime",
              "descripcion": "Categoría para Funkos del anime",
              "tags": [
                "One Piece",
                "Naruto",
                "Hunter x Hunter"
              ],
              "paisOrigen": "Japon",
              "material": "Vinilo"
          }
PUT - /api/categorias/{id-categorias} Modificar una Categoría

Content-Type: application/json - Requiere Token y validacion.

Ejemplo de cuerpo (body) del PUT:

                  {
